# 三行Web速読 (ThreeLinesSokuDoku)

三行Web速読は、ウェブページを効率的に速読するためのネイティブアプリです。画面上に三行ずつ表示し、集中力を維持しながら情報を素早く読み取れます。

## 📝 特長
- 📖 **三行表示:** 読みやすさを追求
- ⏱️ **速読トレーニング:** スピード調整機能
- 🌙 **ダークモード対応**

## 📲 ダウンロード
- [App Store](https://example.com)
- [Google Play](https://example.com)

## 💡 使い方
1. アプリをインストール
2. WebページURLを入力
3. 三行表示モードで速読開始
